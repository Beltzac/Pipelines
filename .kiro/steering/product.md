# Product Overview

**Tugboat Captain's Playground** is a comprehensive desktop application for managing and monitoring TCP (Terminal de Contêineres de Paranaguá) infrastructure. The application provides tools for:

- **Build Management**: Monitor repository builds, view logs, and manage CI/CD pipelines
- **Database Operations**: Compare Oracle schemas, query ESB and SGG systems, backup Oracle views
- **Configuration Management**: Manage Consul key-value pairs, environment configurations
- **Development Tools**: Git repository management, commit tracking, code search
- **System Integration**: Jira integration, automated backups, scheduled jobs

The application is built as a desktop app using Blazor Server with Photino.NET, featuring system tray integration, global hotkeys, and automatic updates. It serves as a centralized dashboard for DevOps and infrastructure management tasks in the TCP environment.

## Key Features

- Multi-environment support (development, staging, production)
- Real-time monitoring and alerting
- Automated backup and synchronization jobs
- Integration with external systems (Jira, Git, Oracle, MongoDB, Consul)
- MCP (Model Context Protocol) server for AI assistant integration
